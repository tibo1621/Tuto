Tuto Page CSS

Introduction

Cette page t’apprend à styliser une page HTML avec du CSS, même sans internet. Chaque partie contient un exemple à copier et un résultat visible.

I. Où mettre le CSS ? (3 méthodes)

  • Interne → dans <style> dans <head>
  • Externe → fichier .css (recommandé)
  • En ligne → style="..." dans la balise

Exemple externe (le nôtre)

<link rel="stylesheet" href="../static/style.css">

Conseil : Toujours utiliser un fichier externe → plus propre !

II. Syntaxe CSS de base

Un sélecteur → une règle → des propriétés :

sélecteur {
    propriété: valeur;
    propriété: valeur;
}

Exemple

h1 {
    color: blue;
    text-align: center;
}

Je suis centré et bleu !

III. Sélecteurs CSS

  • h1 → toutes les balises <h1>
  • .ma-classe → tous les éléments avec class="ma-classe"
  • #mon-id → un seul élément avec id="mon-id"
  • header p → paragraphe dans <header>

Exemple

.titre-rouge { color: red; }
#unique { background: yellow; }
nav a { text-decoration: none; }

V. Marges, padding, bordures (Box Model)

Tout élément = contenu + padding + bordure + marge

Exemple

.boite {
    width: 200px;
    padding: 20px;
    margin: 15px auto;
    border: 3px solid #3498db;
    border-radius: 10px;
    background: #ecf0f1;
    text-align: center;
}
Je suis une boîte !

VIII. Responsive : adapter à mobile

Exemple avec Media Query

@media (max-width: 600px) {
    body {
        font-size: 14px;
        padding: 10px;
    }
    .menu {
        flex-direction: column;
        text-align: center;
    }
}

Test : Rétrécis ta fenêtre → le menu devient vertical !

IX. Glossaire CSS essentiel

  • color → couleur texte
  • background → fond (couleur/image)
  • font-size → taille police (px, rem, em)
  • margin → espace extérieur
  • padding → espace intérieur
  • border → bordure
  • border-radius → coins arrondis
  • display: flex → alignement flexible
  • text-align → alignement texte
  • @media → responsive

X. Projet final : Page complète stylisée

Crée un fichier projet.css et colle :

Clique pour voir le code complet 
/* style.css - Dans static/ */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    background: #f4f7fa;
    color: #333;
    line-height: 1.6;
    padding: 20px;
}

header {
    background: #2c3e50;
    color: white;
    text-align: center;
    padding: 1.5rem;
    border-radius: 10px;
    margin-bottom: 2rem;
}

h1 { font-size: 2.2rem; }
h2 { color: #3498db; margin: 1.5rem 0 1rem; border-bottom: 2px solid #eee; padding-bottom: 0.5rem; }
h3 { color: #e67e22; margin: 1rem 0; }

section {
    background: white;
    padding: 1.5rem;
    margin-bottom: 1.5rem;
    border-radius: 8px;
    box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

pre {
    background: #2c3e50;
    color: #ecf0f1;
    padding: 1rem;
    border-radius: 6px;
    overflow-x: auto;
    font-family: 'Courier New', monospace;
    margin: 1rem 0;
}

code {
    background: #eee;
    padding: 2px 6px;
    border-radius: 4px;
    font-family: monospace;
}

ul {
    margin-left: 1.5rem;
}

footer {
    text-align: center;
    margin-top: 3rem;
    padding: 1rem;
    background: #ecf0f1;
    border-radius: 8px;
}

a {
    color: #2980b9;
    text-decoration: none;
}

a:hover {
    text-decoration: underline;
}

Résultat : Une page moderne, responsive, avec menu, bouton, boîte !
