Tuto Page HTML

Introduction

Cette page est un guide pas à pas pour créer une page web HTML, même sans internet. Utilise-la comme un glossaire ou un tutoriel.

I. Organisation des fichiers

Crée un dossier clair comme ceci :

Dossier principal/
├── static/
│   ├── style.css
│   └── images/
│       └── photo.jpg
│   └── police/
│       └── century_gothic.ttf
├── template/
│   ├── TUTO_HTML.html
│   └── TUTO_CSS.html
└── README.txt

Conseil : Mets tous tes fichiers CSS dans static/ et tes pages dans template/.

II. Créer une page HTML

Crée un fichier nommé index.html dans le dossier template/.

Structure de base (à copier)

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Ma Page</title>
    <link rel="stylesheet" href="../static/style.css">
</head>
<body>
    <h1>Bonjour !</h1>
</body>
</html>

Exemple réel : Ouvre ce fichier → tu verras un titre centré !

III. Structurer sa page : header, body, main, section, div

Utilise les balises sémantiques pour organiser ton contenu :

  • <header> → En-tête (logo, titre)
  • <main> → Contenu principal
  • <section> → Partie thématique
  • <div> → Conteneur générique

Exemple à copier

<header>
    <h1>Mon Site</h1>
</header>
<main>
    <section>
        <h2>Accueil</h2>
        <p>Bienvenue !</p>
    </section>
    <div class="boite">Contenu spécial</div>
</main>
Résultat : Une page bien structurée, prête à être stylisée.

IV. Nommer ces parties pour plus tard affecter un style

Ajoute class ou id pour cibler avec CSS :

  • class="nom" → Pour plusieurs éléments
  • id="nom" → Pour un seul élément

Exemple

<header class="entete">
    <h1 id="titre-principal">Mon Site</h1>
</header>
<div class="carte">Contenu stylisé</div>

En CSS : .carte { background: yellow; } → appliqué à tous les .carte

V. Faire un chemin vers une autre page

Utilise <a href="..."> avec le bon chemin :

Exemple depuis index.html

<a href="page2.html">Aller à la page 2</a>
<a href="../static/images/photo.jpg">Voir l'image</a>
<a href="https://google.com">Google (internet requis)</a>

Chemin relatif : page2.html → même dossier
Chemin absolu : ../static/style.css → remonte d’un dossier

VI. Insérer un bouton

Utilise <button> pour une action ou <a> pour de la navigation :

Exemple

<button>Cliquez-moi !</button>

<a href="page2.html" class="btn">Aller à la page 2</a>
Bouton lien

VII. Insérer une image

Utilise <img> avec src et alt :

Exemple

<img src="../static/images/photo.jpg" alt="Ma photo" width="300">

Conseil : Mets toujours alt pour l’accessibilité.

Si l’image n’existe pas → texte alt s’affiche.

VIII. Faire une barre de navigation

Utilise <nav> + liste :

Exemple complet

<nav>
    <ul class="menu">
        <li><a href="index.html">Accueil</a></li>
        <li><a href="page2.html">Tutoriel CSS</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>
  • Accueil
  • CSS
  • Contact

IX. Balises utiles (glossaire)

  • <h1> → Titre principal (jusqu’à h6)
  • <p> → Paragraphe
  • <img src="..."> → Image
  • <a href="..."> → Lien
  • <div> → Bloc générique
  • <section> → Bloc thématique, un chapitre, un paragraphe
  • <button> → Bouton cliquable
  • <nav> → Barre de navigation
  • <footer> → Pied de page

X. zone de texte

Utilise <input> + placeholder si tu veux en faire plusieurs cote a cote :

Exemple complet

<div<
    <input type="text" placeholder="Zone 1"<
    <input type="text" placeholder="Zone 2"<
    <input type="text" placeholder="Zone 3"<
</div<
